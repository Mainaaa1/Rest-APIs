# Week 9 – REST APIs (9 Mar – 15 Mar 2026)

## Overview

This week focused on building a strong foundation in designing and managing **RESTful APIs**. The learning journey covered core concepts such as API design, versioning, pagination, rate limiting, documentation, and improving existing APIs.

The goal was to move beyond basic endpoint creation and understand how to design APIs that are scalable, maintainable, and production-ready.

---

## Weekly Breakdown

### Day 1: API Design

* Learned REST principles and architecture
* Designed resource-based endpoints
* Used proper HTTP methods and status codes
* Focused on consistent request/response structures

### Day 2: API Versioning

* Explored versioning strategies (URL, header, query)
* Understood backward compatibility
* Learned when and how to version APIs

### Day 3: Pagination

* Implemented offset and cursor-based pagination
* Learned how to handle large datasets efficiently
* Added metadata for better client-side navigation

### Day 4: Rate Limiting

* Studied request throttling techniques
* Explored strategies like token bucket and sliding window
* Learned how to protect APIs from abuse and overload

### Day 5: Swagger Documentation

* Documented APIs using OpenAPI/Swagger
* Learned how to create interactive API docs
* Improved API usability and developer experience

### Day 6: Improve Past APIs

* Refactored existing APIs using best practices
* Added pagination, versioning, and consistent responses
* Improved error handling and structure

### Day 7: Review

* Consolidated all concepts learned during the week
* Reflected on real-world API design considerations
* Identified areas for improvement and deeper exploration

## Reflection

This week emphasized that building APIs is not just about exposing endpoints, but about designing systems that are reliable, predictable, and easy to integrate with.

Understanding these concepts is essential for developing production-grade applications, especially in distributed systems where APIs act as the backbone of communication.
